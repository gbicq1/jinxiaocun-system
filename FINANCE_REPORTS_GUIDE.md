# 财务报表中心使用指南

## 📊 功能介绍

财务报表中心提供五大核心财务报表，帮助您全面了解企业经营状况。

### 报表列表

1. **销售日报** - 每日销售情况统计
2. **销售月报** - 月度经营分析
3. **畅销商品排行** - 商品销售排名
4. **利润分析** - 分类利润统计
5. **库存周转分析** - 库存效率分析

## 🎯 使用流程

### 第一步：进入报表中心

**路径：** 财务管理 → 报表中心

```
左侧菜单 → 财务管理 → 报表中心
```

### 第二步：选择报表类型

在左侧报表列表中选择需要的报表

### 第三步：设置查询条件

1. 选择日期范围
2. 点击 **查询** 按钮

### 第四步：查看报表

在右侧查看报表数据

### 第五步：导出报表

点击 **导出 Excel** 按钮

## 📊 指标说明

### 利润率计算

```
利润率 = (毛利润 / 销售总额) × 100%
毛利润 = 销售总额 - 成本总额
```

### 周转率计算

```
周转率 = (销售数量 / 平均库存) × 100%
平均库存 = (期初库存 + 期末库存) / 2
```

### 周转天数计算

```
周转天数 = 30 / (销售数量 / 平均库存)
```

### 销售占比计算

```
销售占比 = (分类销售额 / 总销售额) × 100%
```

- **销售数据**：销售出库单
- **成本数据**：成本结算模块
- **库存数据**：库存管理模块

### 数据更新

- 实时数据：销售日报、畅销排行
- 定期更新：销售月报、利润分析、库存周转

### 数据精度

- 金额：保留 2 位小数
- 百分比：保留 2 位小数
- 数量：整数

## ❓ 常见问题

### Q1: 报表数据不准确？

**可能原因：**
- 单据未及时保存
- 成本未结算
- 数据未同步

**解决方法：**
1. 检查所有单据是否已保存
2. 执行成本结算
3. 重新查询

### Q2: 某些商品没有数据？

**可能原因：**
- 商品没有销售记录
- 商品已停用
- 查询条件限制

**解决方法：**
1. 检查商品状态
2. 扩大查询范围
3. 确认商品编码

### Q3: 导出数据格式问题？

**解决方法：**
1. 使用 Excel 打开 CSV 文件
2. 设置正确的编码（UTF-8）
3. 调整列宽和格式
