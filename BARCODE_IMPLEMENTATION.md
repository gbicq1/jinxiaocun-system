## 🎯 技术实现

### 工作原理

```
1. 全局监听键盘事件
   ↓
2. 检测快速连续输入（扫描枪特征）
   ↓
3. 缓冲区累积字符
   ↓
4. 检测到回车或超时
   ↓
5. 提交条码并触发事件
   ↓
6. 播放提示音
   ↓
7. 清空缓冲区
```

### 智能识别算法

**扫描枪特征：**
- 快速连续输入（字符间隔 < 100ms）
- 不以 Ctrl/Alt/Shift 开头
- 通常以 Enter 结尾
- 输入内容多为数字或字母组合

**区分手动输入：**
- 字符间隔 > 100ms（可能是手动）
- 单个字符输入
- 包含特殊字符

### 防重复扫描

```typescript
// 记录上次扫描时间
if (lastScanTime && now - lastScanTime < minScanInterval) {
  // 间隔过短，忽略
  return
}
```

1. 进入 **系统设置 → 条码扫描设置**

## 🔧 参数说明

### 扫描超时时间（100ms 推荐）

| 设置值 | 适用场景 |
|--------|---------|
| 50-80ms | 高速扫描枪 |
| 100ms | 标准扫描枪（推荐） |
| 120-150ms | 低速扫描枪 |
| >200ms | 可能是手动输入 |

### 最小扫描间隔（500ms 推荐）

| 设置值 | 效果 |
|--------|------|
| 100-300ms | 允许快速连续扫描 |
| 500ms | 标准防重复（推荐） |
| 800-1000ms | 严格防重复 |
