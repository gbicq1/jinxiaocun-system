## 📊 报表公式

### 利润率
```
利润率 = (毛利润 / 销售总额) × 100%
```

### 周转率
```
周转率 = (销售数量 / 平均库存) × 100%
平均库存 = (期初库存 + 期末库存) / 2
```

### 周转天数
```
周转天数 = 30 / (销售数量 / 平均库存)
```

### 销售占比
```
销售占比 = (分类销售额 / 总销售额) × 100%
```
