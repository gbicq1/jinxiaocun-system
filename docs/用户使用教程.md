## 1. 系统概述

本系统是一款面向中小企业的进销存管理软件，基于 Electron 桌面端架构，支持离线使用，数据存储在本地 SQLite 数据库中，安全可靠。

### 核心功能模块

| 模块 | 功能说明 |
|------|----------|
| 系统首页 | 经营概览、KPI指标、近期单据、库存预警 |
| 期初基础设置 | 产品、仓库、供应商、客户、价格表管理 |
| 采购管理 | 采购入库、采购退货、采购开票管理 |
| 销售管理 | 销售出库、销售退货、销售开票管理 |
| 库存管理 | 实时库存查询、库存调拨、库存期初期末 |
| 财务管理 | 对账、成本结算、收款单、付款单、报表中心 |
| 系统设置 | 用户管理、角色管理、职工档案、条码设置、操作日志、回收站、备份恢复 |

## 2. 快速入门

### 2.1 启动系统

双击桌面快捷方式或在项目目录下运行启动脚本，系统将自动打开主界面。

### 2.2 首次使用流程

首次使用系统时，建议按以下顺序完成初始化：

```
第一步：期初基础设置
  ├── 录入产品基础数据
  ├── 录入仓库仓位数据
  ├── 录入供应商信息
  └──  录入客户信息
  

第二步：日常业务操作
  ├── 采购入库 → 库存增加
  ├── 销售出库 → 库存减少
  └── 退货处理 → 库存冲减

第三步：财务核算
  ├── 收款/付款登记
  ├── 供应商/客户对账
  └── 月末成本结算
```

### 2.3 界面布局说明

系统采用经典的左侧菜单 + 右侧内容区布局：

- **左侧导航栏**：按功能模块分组，点击菜单项切换页面
- **顶部标题栏**：显示当前页面名称
- **主内容区**：展示当前模块的操作界面

## 3. 期初基础设置

> ⚠️ **重要提示**：基础数据是系统正常运行的前提，请务必在使用业务功能前完成基础数据的录入。

### 3.1 产品基础数据

**功能说明**：管理所有产品的基础信息，是整个系统的核心数据。

**操作步骤**：

1. 点击左侧菜单 **期初基础设置 → 产品基础数据**
2. 点击 **新增** 按钮，打开新增对话框
3. 填写产品信息：
   - **产品编码**：唯一标识，不可重复（如：A001）
   - **产品名称**：产品的中文名称
   - **分类**：产品所属类别（如：粮油、水果）
   - **单位**：计量单位（如：个、箱、吨）
   - **条码**：产品条形码，支持扫码枪输入（暂未开发）
   - **规格型号**：产品的规格描述
   - **税率**：默认税率（免税/1%/3%/5%/6%/9%/13%），选择后后期填写单据时可以避免手动选择
   - **成本价**：产品的采购成本价（暂未开发，不用填写）
   - **销售价**：产品的默认销售价（暂未开发，不用填写）
   - **预警数量**：库存低于此数量时系统预警
4. 点击 **保存** 完成新增

**其他操作**：
- **编辑**：点击列表中的编辑按钮修改产品信息
- **删除**：点击删除按钮移除产品（状态为停用的产品不会参与业务）
- **导出**：将产品列表导出为 Excel 文件
- **搜索**：支持按编码、名称、分类等条件搜索

### 3.2 仓库仓位数据

**功能说明**：管理仓库信息，支持多仓库管理。

**字段说明**：
- **仓库编码**：唯一标识
- **仓库名称**：如"农服仓库"、"驿站仓库"
- **地址**：仓库的物理地址
- **联系人/联系电话**：仓库管理员信息

### 3.3 供应商信息

**功能说明**：管理供应商档案，用于采购业务关联。

**字段说明**：
- **供应商编码/名称**
- **联系人/联系电话/联系邮箱**
- **地址**
- **开户银行/银行账号**：用于付款时参考

### 3.4 客户信息

**功能说明**：管理客户档案，用于销售业务关联。

**字段说明**：
- **客户编码/名称**
- **联系人/联系电话/联系邮箱**
- **地址**
- **信用额度**：客户赊销上限（暂未开发，不用填写）

### 3.5 供应商价格表（暂未开发，不用填写）

**功能说明**：维护各供应商对不同产品的供货价格，方便采购时快速引用。

**操作说明**：
- 选择供应商后，可批量设置各产品的采购价和销售价
- 支持设置价格生效日期
- 价格表数据在创建采购入库单时可自动引用

## 4. 采购管理

### 4.1 采购入库

**功能说明**：记录采购商品的入库信息，入库后库存自动增加。

**新增入库单流程**：

1. 点击 **新增入库单** 按钮
2. 填写单据头部信息：
   - **凭证号**：自动生成，格式为 `RK+日期+随机数`
   - **入库日期**：选择实际入库日期
   - **供应商**：选择供应商（必填）
   - **仓库**：选择入库仓库（必填）
3. 添加商品明细：
   - 点击商品下拉框选择商品
   - 填写数量、不含税单价、税率
   - 系统自动计算：含税单价、税额、金额
4. 设置发票状态：
   - 切换"已开票/未开票"开关
   - 选择发票类型（普票/专票）
   - 普票+免税商品可开启"加计扣除"
5. 点击 **保存** 完成入库

**价格计算规则**：
- 输入不含税单价 → 自动计算含税单价 = 不含税单价 × (1 + 税率%)
- 输入含税单价 → 自动反算不含税单价
- 金额 = 含税单价 × 数量
- 税额 = 金额 - 不含税单价 × 数量

**加计扣除说明**：
- 仅当发票状态为"已开票"、发票类型为"普票"、税率为"免税"时可用
- 加计扣除金额 = 金额 × 9%

**查询与筛选**：
- 支持按日期范围、供应商、金额范围筛选
- 支持导出 Excel 和打印

### 4.2 采购退货

**功能说明**：处理向供应商退回商品的业务，退货后库存自动减少。

**新增退货单流程**：

1. 点击 **新增退货单**
2. 选择 **原入库单号**：系统自动加载原入库单的商品列表和价格
3. 在商品列表中填写退货数量（不能超过原入库数量）
4. 填写退货原因（必填）
5. 选择经办人
6. 点击 **保存**

**校验规则**：
- 退货数量不能超过原入库单中对应商品的数量
- 同一入库单的多次退货，累计退货数量不能超过原入库数量
- 有关联退货单的入库单不能直接删除
- 修改入库单数量时，修改后数量不能小于已退货数量

### 4.3 采购开票管理

**功能说明**：管理采购入库单的开票状态和开票金额。

**操作说明**：
- 查看所有入库单的开票状态
- 可按单据标记"已开票/未开票"
- 记录开票金额和开票日期
- 支持分次开票（一张入库单可多次开票）

## 5. 销售管理

**新增出库单流程**：

2. 填写单据信息：
   - **凭证号**：自动生成，格式为 `CK+日期+随机数`

   - **客户**：选择客户（必填）

3. 添加商品明细：

**库存校验**：

### 5.3 销售开票管理

**功能说明**：管理销售出库单的开票状态。

**操作说明**：
- 查看所有出库单的开票状态
- 标记开票状态和开票金额
- 支持按客户、日期筛选

## 6. 库存管理

### 6.1 实时库存查询

**功能说明**：查看各仓库中各产品的实时库存数量和成本。

**操作说明**：
- 系统自动汇总所有入库、出库、退货、调拨数据计算实时库存
- 支持按仓库、产品分类筛选
- 显示字段：产品编码、名称、规格、仓库、库存数量、成本价、预警状态
- 库存数量低于预警值时以红色标记
- 支持点击产品查看库存明细账（台账）
- 支持导出 Excel

**库存明细账**：
- 按时间顺序展示某产品在某仓库的所有出入库记录
- 包含：日期、单号、类型、入库数量/金额、出库数量/金额、结存
- 支持按日期范围筛选

### 6.2 库存调拨

**功能说明**：在不同仓库之间调拨商品。

**新增调拨单流程**：

1. 点击 **新增调拨单**
2. 选择 **调出仓库** 和 **调入仓库**
3. 添加调拨商品和数量
4. 系统自动带出当前成本价
5. 点击 **保存**

**校验规则**：
- 调出仓库的库存必须充足
- 调出仓库和调入仓库不能相同

### 6.3 库存期初期末

**功能说明**：记录各产品在各仓库的期初库存，用于月末盘点和核算。

**操作说明**：
- 可手动录入期初数据
- 系统根据出入库记录自动计算期末数据
- 支持按期间查询

## 7. 财务管理

### 7.1 供应商/客户对账

**功能说明**：与供应商或客户进行往来账目核对。

**操作说明**：
- 选择对账对象类型（供应商/客户）
- 选择具体的供应商或客户
- 设置对账日期范围
- 系统自动汇总该期间内的所有入库/出库、退货、收付款记录
- 显示应收/应付余额
- 支持导出对账单

### 7.2 成本结算

**功能说明**：按月进行库存成本核算，采用加权平均法计算产品成本。

**操作流程**：

1. **设置启用月份**：首次使用需设置成本结算的起始月份
2. **选择结算月份**：在年度月份面板中点击要结算的月份
3. **查询数据**：设置日期范围、产品、仓库等筛选条件
4. **开始计算**：点击"开始计算"按钮，系统自动按加权平均法计算各产品的月度成本
5. **确认结算**：计算完成后月份状态变为"已结算"

**加权平均法说明**：
- 本月加权平均成本 = (期初库存金额 + 本月入库金额) ÷ (期初库存数量 + 本月入库数量)
- 出库成本 = 出库数量 × 加权平均成本
- 期末库存金额 = 期末库存数量 × 加权平均成本

**反结算**：
- 已结算的月份可以点击"反结算"撤销结算
- 反结算后该月份的数据可以修改
- ⚠️ 只能从最近一个已结算月份开始反结算

**结算锁定规则**：
- 已结算月份的单据不能新增、修改、删除
- 需先反结算才能操作该月份的单据

**导出功能**：
- 导出汇总：导出月度成本汇总表
- 导出明细账：导出各产品的出入库明细账

### 7.3 收款单

**功能说明**：记录从客户处收到的款项。

**字段说明**：
- **收款单号**：自动生成
- **客户**：选择收款客户
- **收款日期**
- **收款金额**
- **收款方式**：现金/转账/支票等
- **备注**

### 7.4 付款单

**功能说明**：记录向供应商支付的款项。

**字段说明**：
- **付款单号**：自动生成
- **供应商**：选择付款供应商
- **付款日期**
- **付款金额**
- **付款方式**
- **备注**

### 7.5 报表中心

**功能说明**：提供各类经营报表的查询和导出。

**报表类型**：
- 销售汇总报表
- 采购汇总报表
- 库存报表
- 收付款报表
- 利润分析报表

所有报表均支持导出 Excel 和打印。

**功能说明**：记录系统中所有重要操作的日志。

**备份操作**：

2. 系统自动将当前数据库文件复制到备份目录

1. 从备份列表中选择一个备份点
2. 点击"恢复"按钮
3. ⚠️ 恢复操作将覆盖当前数据，请谨慎操作

**自动备份**：
- 系统支持定时自动备份
- 可设置备份保留天数

## 9. 业务规则与校验说明

### 9.1 单据编号规则

| 单据类型 | 编号前缀 | 格式示例 |
|----------|----------|----------|
| 采购入库单 | RK | RK202604210001 |
| 销售出库单 | CK | CK202604210001 |
| 采购退货单 | CH | CH202604210001 |
| 销售退货单 | XS | XS202604210001 |
| 库存调拨单 | DB | DB202604210001 |

### 9.2 退货业务规则

1. **退货数量限制**：退货数量不能超过原单据中对应商品的数量
2. **累计退货限制**：同一原单据的多次退货，累计退货数量不能超过原单据数量
3. **原单据删除保护**：有关联退货单的原单据（入库单/出库单）不能删除，必须先删除所有关联的退货单
4. **原单据修改保护**：修改入库单/出库单的商品数量时，修改后数量不能小于已累计退货数量

**错误提示示例**：
- 删除入库单时：`无法删除入库单！该入库单存在关联的采购退货单：CH202604210001，请先删除相关退货单`
- 修改数量时：`无法保存！商品"苹果"修改后数量(50)小于退货单累计退货数量(80)`

### 9.3 成本结算锁定规则

1. 已结算月份的单据不能新增、修改、删除
2. 需先反结算才能操作该月份的单据
3. 反结算只能从最近一个已结算月份开始，不能跳月反结算
4. 成本结算采用加权平均法

### 9.4 库存计算规则

实时库存 = 采购入库数量 + 销售退货数量 + 调拨入库数量 - 销售出库数量 - 采购退货数量 - 调拨出库数量

### 9.5 价格计算规则

```
含税单价 = 不含税单价 × (1 + 税率/100)
不含税单价 = 含税单价 ÷ (1 + 税率/100)
金额(含税) = 含税单价 × 数量
税额 = 金额(含税) - 不含税单价 × 数量
```

**加计扣除（特殊模式）**：
- 适用条件：已开票 + 普票 + 免税
- 加计扣除金额 = 金额 × 9%
- 不含税单价 = 含税单价 × (1 - 9%)

## 10. 常见问题

### Q1：如何修改已结算月份的单据？

**A**：需要先进行"反结算"操作。进入 **财务管理 → 成本结算**，选择已结算的月份，点击"反结算"按钮，撤销结算后即可修改该月份的单据。修改完成后需重新进行成本结算。

### Q2：删除入库单/出库单时提示"存在关联的退货单"怎么办？

**A**：系统不允许直接删除有关联退货单的原单据。请先进入退货单管理页面，删除所有关联的退货单后，再删除原单据。

### Q3：修改入库单/出库单数量时提示"修改后数量小于退货单数量"怎么办？

**A**：这是系统的保护机制。如果已存在退货单，原单据的修改后数量必须大于或等于累计退货数量。您可以：
- 先删除相关退货单，再修改原单据
- 将原单据数量修改为大于累计退货数量的值

### Q4：库存数量为负数怎么办？

**A**：库存为负通常是因为出库数量超过了入库数量。请检查是否有遗漏的入库单未录入，或者出库数量是否填写错误。

### Q5：如何设置默认仓库和默认经办人？

**A**：在新增单据时，仓库和经办人下拉框旁边有"默认"开关，点击即可将该选项设为默认值，后续新增单据时会自动填充。

### Q6：数据如何备份？

**A**：进入 **系统设置 → 备份与恢复**，点击"创建备份"即可手动备份。系统也支持定时自动备份功能。

### Q7：误删除的单据如何恢复？

**A**：进入 **系统设置 → 回收站**，找到误删除的单据，点击"恢复"即可还原。

### Q8：如何使用条码扫描功能？

**A**：进入 **系统设置 → 条码扫描设置**，配置扫描枪的串口和波特率等参数，启用扫描功能后，在商品选择界面扫描商品条码即可自动匹配商品。

### Q9：凭证号可以手动修改吗？

**A**：凭证号由系统自动生成，建议不要手动修改，以保持编号的唯一性和连续性。

### Q10：如何导出数据？

**A**：大部分列表页面都提供"导出 Excel"按钮。也可以在报表中心导出各类汇总报表。成本结算页面支持导出成本汇总表和明细账。
